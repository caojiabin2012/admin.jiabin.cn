ReflectionExtension::isTemporary

(PHP >= 5.4.0)

ReflectionExtension::isTemporary — 返回扩展是否是临时载入

说明

public void ReflectionExtension::isTemporary ( void )

Warning

本函数还未编写文档，仅有参数列表。

参数

此函数没有参数。

返回值

如果扩展被dl()载入则返回TRUE ，否则返回 FALSE 。

参见

  • ReflectionExtension::isPersistent() - 返回扩展是否持久化载入
