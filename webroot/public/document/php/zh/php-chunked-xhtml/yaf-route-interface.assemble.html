Yaf_Route_Interface::assemble

(Yaf >=2.3.0)

Yaf_Route_Interface::assemble — 将指定路由规则组合成一个url

说明

abstract public string Yaf_Route_Interface::assemble ( array $info [, array $query ] )

这个方法会将指定的参数加上自定义的参数组合成一个url

所有route必须结合其路由规则来实现这个方法

参数

info

query
