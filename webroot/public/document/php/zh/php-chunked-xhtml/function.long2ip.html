long2ip

(PHP 4, PHP 5, PHP 7)

long2ip — 将长整型转化为字符串形式带点的互联网标准格式地址（IPV4）

说明

string long2ip ( string $proper_address )

long2ip() 函数通过长整型的表达形式转化生成带点格式的互联网地址（例如：aaa.bbb.ccc.ddd ）。

参数

proper_address

合格的地址，长整型的表达形式。

返回值

返回字符串的互联网 IP 地址。

参见

  • ip2long() - 将一个IPV4的字符串互联网协议转换成数字格式
