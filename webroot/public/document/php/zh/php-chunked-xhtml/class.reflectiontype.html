The ReflectionType class

(PHP 7)

简介

ReflectionType 类用于获取函数、类方法的参数或者返回值的类型。

类摘要

ReflectionType {
/* 方法 */
public bool allowsNull ( void )
public bool isBuiltin ( void )
public string __toString ( void )
}
