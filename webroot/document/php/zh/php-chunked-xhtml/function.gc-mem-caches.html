gc_mem_caches

(PHP 7)

gc_mem_caches — Reclaims memory used by the Zend Engine memory manager

说明

int gc_mem_caches ( void )

Reclaims memory used by the Zend Engine memory manager.

参数

此函数没有参数。

返回值

Returns the number of bytes freed.
