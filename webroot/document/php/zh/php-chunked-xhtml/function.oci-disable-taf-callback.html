oci_disable_taf_callback

(PHP >= 7.0.21, PHP >= 7.1.7, PHP 7.2, PECL OCI8 >= 2.1.5)

oci_disable_taf_callback — Disable a user-defined callback function for Oracle Database TAF

说明

bool oci_disable_taf_callback ( resource $connection )

Disables the user-defined callback function registered to connection by oci_register_taf_callback(). See OCI8 Transparent Application Failover (TAF) Support for information about OCI8 TAF support.

Use oci_register_taf_callback() to register a user-defined callback.

参数

connection

An Oracle connection identifier.

返回值

成功时返回 TRUE， 或者在失败时返回 FALSE。
