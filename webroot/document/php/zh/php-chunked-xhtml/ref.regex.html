Warning

This feature was DEPRECATED in PHP 5.3.0, and REMOVED in PHP 7.0.0.

Alternatives to this feature include:

  • PCRE （支持完整的正则表达式）
  • fnmatch() （支持 shell 风格通配符的匹配）
